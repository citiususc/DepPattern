Unary relationship

The types of dependencies described above are used to identify word dependencies. However, DepPattern also permits to identify a PoS tagg in context to make different operations on it: morpho-syntactic corrections, addition of semantic or pragmatic information, modification of some features, etc. For this purpose, we defined a unary relation type: 'Head'. In Section 2.2 of this tutorial, we show an example of how this type of relation is used to solve systematic PoS tagging errors. In addition, in Section 2.3, it is used to set function unicity.
